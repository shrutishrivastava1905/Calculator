# 🧮 Simple Calculator (C# Console App)

A beginner-friendly **Calculator application** built using C#. This project performs basic arithmetic operations through a console-based interface.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* ⚠️ Error handling (e.g., divide by zero)

---

## 🛠️ Technologies Used

* C#
* .NET Console Application

---

## 📁 Project Structure

```
CalculatorApp/
│
├── Program.cs
├── CalculatorApp.csproj
├── bin/
└── obj/
```

---

## ▶️ How to Run

### 🔹 Using Visual Studio

1. Open the project
2. Press **F5** or click **Run ▶**

### 🔹 Using .NET CLI

```bash
dotnet new console -n CalculatorApp
cd CalculatorApp
dotnet run
```

---

## 💡 How It Works

1. User enters two numbers
2. User selects an operation (+, -, *, /)
3. Program calculates the result
4. Displays output in console

---

## 📌 Example

```
=== Simple Calculator ===
Enter first number: 10
Enter second number: 5
Choose operation: +
Result = 15
```

---

## ⚠️ Error Handling

* Prevents division by zero
